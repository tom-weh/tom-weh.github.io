Languages

In Web Design, there are different languages for different purposes.

HTML5

In long it is called Hypertext Markup Language and it is used to markup. The general structure and content is usually located in this file. Good HTML-code follows the POUR principle.

CSS3

In long it is called Cascading Style Sheets and it is used to style a webpage (as the name also implies). There are also Frameworks built in css which lets you use predefined styles and apply them via the HTML class attribute

Javascript

In short it is just called JS and it is used to give a webpage functionality. It is used to transfer forms for example but it can also be used to change the HTML. There is almost no limits in what you can do with it.

Tailwind Framework

First of all: A Frameworks is not a language like HTML, CSS or Javascript. It is a library using CSS to support Web Designers to create websites faster. Sometimes, Frameworks also includes JavaScript. But primarily they are used for styling purposes.

There are many frameworks out there. Bootstrap is one of the most popular. However the use of a framework like Bootstrap often times results in similar looking websites. This can be good as the user is familiar to the general structure and feeling, but it can also be boring. To be able to create a unique website I decided to use the Tailwind Framework

Tailwind is a low-level framework which is highly customizable. It is built to create good looking websites without spending too much time on basic styling with CSS from scratch.
